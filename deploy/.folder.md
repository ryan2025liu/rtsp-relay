# Folder: /deploy

1. **地位 (Position)**: 部署资产目录，管理容器、systemd、nginx 运行模板。
2. **逻辑 (Logic)**: 为单机服务器提供统一部署入口和生产运行模板。
3. **约束 (Constraints)**: 不放业务源码；部署变更必须同步 `docs/03-operations`。
4. **资产 (Assets)**: `env.server.example` 与 `docs/03-operations/deploy-lan-server.md` 搭配根目录 `docker-compose.server.yml`、`apps/web/Dockerfile.prod` 使用。
